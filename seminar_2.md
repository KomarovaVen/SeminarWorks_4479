## Заголовки

Чтобы добавить заголовки, используйте символ (#).
Виды заголовков:

```
# Заголовок первого уровня #
## Заголовок h2
### Заголовок h3
#### Заголовок h4
##### Заголовок h5
###### Заголовок h6

```

В декоративных целях заголовки можно «закрывать» с обратной стороны.
